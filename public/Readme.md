## ✨ Features

| Feature | Description |
|---------|-------------|
| 📱 **Multi-Device** | Connect multiple devices simultaneously |
| 🎨 **QR & Pairing** | Two login methods — QR Code or Phone Number |
| 🌐 **Web Dashboard** | Real-time control panel with live logs |
| ⚡ **Fast Response** | Lightning-fast command processing |
| 🔒 **Secure** | Owner-only commands with authorization |
| 🎯 **Easy Commands** | Simple commands like `.ping`, `.help`, `.info` |
| 💻 **Cross-Platform** | Works on Linux, Windows, macOS |
| 🔄 **Auto-Reconnect** | Automatically reconnects on disconnect |
| 📊 **Live Logs** | Real-time connection logs on dashboard |
| 🎨 **Dark/Light Mode** | Theme support on web dashboard |

---

## 📋 Available Commands

| Command | Description | Example |
|---------|-------------|---------|
| `ping` / `.ping` | Check bot status | `ping` |
| `help` / `.help` / `/help` / `menu` | Show help menu | `.help` |
| `info` / `.info` / `/info` | Show bot information | `.info` |

## 🚀 Quick Start

### Prerequisites

- Node.js 18.x or higher
- npm or yarn
- A WhatsApp account

### Installation

```bash
# Clone the repository
git clone https://github.com/charukamahesh922-collab/Zara---MD.git

# Navigate to the directory
cd Zara---MD

# Install dependencies
npm install

# Install image processing library (required)
npm install sharp

# Start the bot
node index.js
